Allow customers to contact you using Email

== Description ==

Don't miss any potential customers!
Make it easier for your customers to contact you! Make your email contact much more accessible for your users, so they can easily contact you.
With 'Email - Contact Us' you can add a small icon at the corner with your email address and short message as "Contact us" or "message us" etc. When the user clicks on the icon - it will open his default email client.
Smartarget Email Contact Us plugin works also with WooCommerce, Elementor and Oxygen Builder.

== Installation ==

1. Unpack plugin zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open plugin settings page
4. Configure settings and press Save
